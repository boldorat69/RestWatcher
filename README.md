# 📊 Telegram-бот аналитики продаж для r_keeper

**Хватит тратить время на сложную и медленную админку r_keeper.** Получайте чистую, понятную и мгновенную бизнес-аналитику вашего заведения прямо в Telegram — в один клик.

---

## 🚀 Что умеет бот?

-   **⚡ Мгновенные отчеты:** Больше не нужно заходить в систему с компьютера, бот выплевывает ключевые цифры за секунды.
-   **📅 Любые периоды (Новинка!):** Выбирайте отчет за 1 день, текущую неделю, 2 недели или за весь прошлый месяц.
-   **📉 Сравнение продаж:** Главная фишка — возможность задать *любой свой промежуток дат* и мгновенно сравнить его с аналогичным предыдущим периодом. Вы сразу увидите динамику выручки бизнеса.
-   **🛡️ Безопасность:** Прямое и защищенное подключение к SQL-базе данных вашей системы.
