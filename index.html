Возможности

RestWatcher предоставляет полный набор инструментов для анализа работы заведения прямо в Telegram. Большинство отчётов поддерживают произвольный диапазон дат и сравнение с предыдущим периодом.

📈

Продажи

  • Сводка продаж — выручка, средний чек, количество гостей и чеков за период. Детализация по дням с просмотром каждого дня
  • Топ 50 блюд — рейтинг блюд по выручке и количеству продаж с пагинацией
  • Топ категорий — выручка по категориям меню
  • Топ 10 столов — рейтинг столов по выручке, количеству чеков и среднему чеку
  • Заказы по категориям — детальная разбивка: категория → подкатегория → блюдо, с типами оплат
  • Выручка по часам — почасовая разбивка с пиковыми часами и сегментами (завтрак/обед/ужин)
  • Специальные отчёты — произвольные отчёты под конкретные требования
🏷️

Скидки и наценки

  • Сводка по скидкам — общая сумма скидок, наценок и итог после применения за период. Детализация по дням
  • Детальный отчёт — разбор скидок и наценок по типам для выбранной даты
❌

Отмены

  • Сводка по отменам пречеков — количество и сумма отменённых пречеков за период. Детализация по дням с максимумами
  • Детали отмен — список отменённых пречеков с указанием времени, стола, менеджера и причины
  • Топ менеджеров — рейтинг менеджеров по количеству и сумме отмен
👥

Персонал

  • Топ официантов — рейтинг по выручке за период. Детализация по дням
  • Топ менеджеров по отменам — контроль качества работы управляющего персонала
🔄

Смены

  • Сводка по смене — выручка по типам оплат (наличные/карта/неплательщики), количество чеков и гостей
  • Чеки за дату — список всех чеков с фильтрацией по столу и просмотром содержимого
  • Открытая смена — мониторинг текущей смены в реальном времени: продажи, скидки, отмены, топ столов
📅

Выбор периода

  • Пресеты — «Вчера», «Последние 7/14 дней», «Этот/Прошлый месяц»
  • Ручной ввод — произвольный диапазон дат в формате ДД.ММ.ГГГГ
  • Сравнение периодов — каждый отчёт можно сравнить с предыдущим периодом с дельтой и процентом изменения
🤖

ИИ-ассистент

  • Задайте вопрос голосом 🎤 или текстом ✍️ на русском, украинском или английском
  • ИИ самостоятельно анализирует запрос, подбирает таблицы и связи из базы знаний r_keeper, генерирует SQL
  • Результат форматируется в читаемый отчёт с итогами, долями и пиковыми значениями
  • Поддержка Gemini AI (основной) и GPT-4o (фолбэк)
  • Работает через read-only пользователя БД — безопасность данных гарантирована

Как это работает

Всё, что нужно для начала работы — пара простых шагов.

1

Установка

Добавьте бота в Telegram и получите доступ от администратора вашего ресторана.

2

Настройка

Администратор подключает ваш ресторан через веб-панель. База данных остаётся защищённой.

3

Отчёты

Пользуйтесь готовыми отчётами по продажам, отменам, персоналу и задавайте вопросы ИИ-ассистенту.

Безопасность

RestWatcher спроектирован с учётом приоритета безопасности данных вашего ресторана.

🔒

Read-only доступ к базе данных

Все запросы выполняются только на чтение. Никаких изменений в данных.

🔐

AES-256 шифрование

Весь трафик между ботом и агентом шифруется. Перехват данных невозможен.

📋

Полный аудит

Каждое действие пользователей логируется. Всегда можно отследить кто и что запрашивал.

🚫

Бан по IP

При неудачных попытках авторизации агента IP блокируется на 10 минут.
