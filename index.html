Возможности

RestWatcher предоставляет управляющим ресторанов полный набор инструментов для анализа работы заведения прямо в Telegram. Все отчёты поддерживают произвольный диапазон дат и сравнение с предыдущим периодом.

📊

Продажи

  • Сводка продаж — выручка, средний чек, количество гостей и чеков за период. Детализация по дням с просмотром каждого дня
  • Топ блюд — рейтинг блюд по выручке и количеству продаж с пагинацией
  • Топ категорий — выручка по категориям меню (основная классификация 2560)
  • Топ столов — рейтинг столов по выручке, количеству чеков и среднему чеку
  • Заказы по категориям — детальная разбивка: категория → подкатегория → блюдо, с типами оплат
  • Выручка по часам — почасовая разбивка с пиковыми часами 🔥 и сегментами (завтрак/обед/ужин)
  • Специальный отчёт — кастомная сводка: выручка, количество, средний чек с дельтой
🏷️

Скидки и наценки

  • Сводка по скидкам — общая сумма скидок, наценок и итог после применения за период. Детализация по дням
  • Детальный отчёт — разбор скидок и наценок по типам для выбранной даты
🚫

Отмены

  • Сводка по отменам — количество и сумма отменённых пречеков за период. Детализация по дням с максимумами
  • Детали отмен — список отменённых пречеков с указанием времени, стола, менеджера и причины
  • Топ менеджеров по отменам — рейтинг менеджеров по количеству и сумме отмен
👨‍🍳

Персонал

  • Топ официантов — рейтинг по выручке за период. Детализация по дням
  • Топ менеджеров по отменам — контроль качества работы управляющего персонала
🔄

Смены

  • Сводка по смене — выручка по типам оплат (нал/карта), количество чеков и гостей за дату
  • Чеки за дату — список всех чеков с возможностью фильтрации по номеру стола и просмотра деталей каждого чека
  • Открытая смена — мониторинг текущей смены в реальном времени: продажи, скидки, отмены, чеки, топ столов
📅

Гибкий выбор периода

  • Пресеты — «Вчера», «Позавчера», «Последние 7 дней», «Последние 14 дней», «Этот месяц», «Прошлый месяц»
  • Ручной ввод — произвольный диапазон дат в формате ДД.ММ.ГГГГ
  • Сравнение периодов — каждый отчёт можно сравнить с предыдущим периодом той же длины с дельтой и процентом изменения
🤖

ИИ-ассистент

  • Задайте вопрос голосом 🎤 или текстом ✍️ на русском, украинском или английском
  • ИИ самостоятельно анализирует запрос, подбирает таблицы и связи из базы знаний r_keeper, генерирует SQL
  • Результат форматируется в читаемый отчёт с итогами, долями и пиковыми значениями
  • Поддержка Gemini AI (основной) и GPT-4o (фолбэк) для отказоустойчивости
  • Работает через read-only пользователя БД — безопасность данных гарантирована
